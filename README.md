# Terminal Cheatsheet

### Logging in

ssh <sunetid>@login.sherlock.stanford.edu

### Edit files

`vi file.ext`

to edit `i`

when done, `esc` then `:x`


### Running .sh

`sbatch file.sh`

`squeue -u $USER`

`cat file.out`



torch is big to install, so use `pip install torch --no-cache-dir`
